# Histórias de Usuário com Funcionalidades

## 1. Reservar um Quarto
- **Como cliente**, quero selecionar um hotel e reservar um quarto para datas específicas, para garantir minha hospedagem.  
  - **Critérios de Aceitação**:
    - Escolher datas disponíveis.
    - Confirmar reserva com detalhes de pagamento.

## 2. Cancelar Reserva
- **Como cliente**, quero poder cancelar minha reserva, para ter flexibilidade em meus planos.  
  - **Critérios de Aceitação**:
    - Opção de cancelamento com confirmação.
    - Política de cancelamento visível.

## 3. Autenticar Usuário
- **Como cliente**, quero me registrar e fazer login, para acessar minha área pessoal.  
  - **Critérios de Aceitação**:
    - Validação de email e senha.
    - Mensagens de erro claras.

## 4. Gerenciar Quartos (Admin)
- **Como administrador**, quero adicionar ou remover quartos, para manter o inventário atualizado.  
  - **Critérios de Aceitação**:
    - Formulário para adicionar/editar quartos.
    - Visualização da lista de quartos.
